<objective>
Fork litgpt/pretrain.py into safemoe/pretrain.py with the SGTM dual-optimizer
3-path training loop. This is the core Phase 3 deliverable: replaces litgpt's
single optimizer with optimizer_harmful + optimizer_std (each over their
respective parameter sets from HarmfulParamRegistry), replaces the single
CycleIterator with three split iterators + weighted split sampling, and wraps
the inner accumulation loop in if/elif/else branching that calls the correct
masker for each split label.

This plan also completes the attn-head activation masking deferred from 03-01:
SafeCausalSelfAttention.forward implements the actual head-output zeroing
(attn_out[:, head_idx, :, :] = 0 before reshape+proj) that the flag
infrastructure in masking.py controls.

Purpose: Enables training a SafeMoE model on three data streams (D_std,
D_harmful, D_unlabeled) with gradient and activation isolation enforced per
step. TRAIN-01 and TRAIN-02 requirements satisfied.

Output: safemoe/pretrain.py (fork of litgpt/pretrain.py, ~400-500 lines);
unit tests test_fit_*_step PASS GREEN.
</objective>

# Split sampling at optimizer step start:
SPLIT_LABELS = ["D_std", "D_harmful", "D_unlabeled"]
weights = [train.upsample_std, train.upsample_harmful, train.upsample_unlabeled]
split_label = random.choices(SPLIT_LABELS, weights=weights, k=1)[0]

# Three-path masking (from RESEARCH.md Pattern 2):
if split_label == "D_std":
    activation_masker.enable()
try:
    for micro_batch_idx in range(accum_iters):
        is_accumulating = micro_batch_idx < accum_iters - 1
        with fabric.no_backward_sync(model, enabled=is_accumulating):
            logits = model(input_ids)
            loss = chunked_cross_entropy(logits, targets) / accum_iters
            fabric.backward(loss)
finally:
    if split_label == "D_std":
        activation_masker.disable()

if split_label == "D_std":
    fabric.clip_gradients(model, optimizer_std, max_norm=train.max_norm)
    optimizer_std.step(); optimizer_std.zero_grad(set_to_none=True)
    optimizer_harmful.zero_grad(set_to_none=True)
elif split_label == "D_harmful":
    gradient_masker.mask()
    fabric.clip_gradients(model, optimizer_harmful, max_norm=train.max_norm)
    optimizer_harmful.step(); optimizer_harmful.zero_grad(set_to_none=True)
    optimizer_std.zero_grad(set_to_none=True)
else:  # D_unlabeled
    fabric.clip_gradients(model, optimizer_harmful, max_norm=train.max_norm)
    fabric.clip_gradients(model, optimizer_std, max_norm=train.max_norm)
    optimizer_harmful.step(); optimizer_std.step()
    optimizer_harmful.zero_grad(set_to_none=True); optimizer_std.zero_grad(set_to_none=True)

# Dual optimizer setup (from RESEARCH.md Pattern 3):
registry = HarmfulParamRegistry(model, config)
extra_kwargs = {"fused": fabric.device.type == "cuda"}
optimizer_harmful = instantiate_torch_optimizer(optimizer, registry.parameters_by_type("theta_harmful"), **extra_kwargs)
optimizer_std = instantiate_torch_optimizer(optimizer, registry.parameters_by_type("theta_std"), **extra_kwargs)
optimizer_harmful, optimizer_std = fabric.setup_optimizers(optimizer_harmful, optimizer_std)

# Dual LR update (Pattern 4):
lr = get_lr(base_lr, state["iter_num"], warmup_iters, max_iters, train.min_lr)
