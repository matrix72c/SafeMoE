<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
    Complete Phase 4 evaluation pipeline:
    - ablate CLI (TRAIN-04): zeros theta_harmful in a checkpoint, saves ablated/ subdir + manifest
    - evaluate CLI (EVAL-01): per-split perplexity comparison table + results.json
    - evaluate --routing (EVAL-02): per-split routing fraction histograms + routing_attribution.json
    - evaluate_with_ablation() in pretrain.py (EVAL-03): mid-training ablation metrics in TensorBoard
  </what-built>
  <how-to-verify>
    **Prerequisites:** A trained SafeMoE checkpoint (run at least a few hundred steps with
    `python -m safemoe pretrain --config safemoe/configs/safemoe-tinystories.yaml`).
    Set CKPT_DIR to the checkpoint directory (e.g., `out/safemoe-tinystories/step-00000200/`).

    **Step 1 — Ablation (TRAIN-04):**
    ```bash
    python -m safemoe ablate $CKPT_DIR
    ```
    Expected:
    - Prints summary table with "Parameters zeroed", "Total norm (before) > 0", "Total norm (after) = 0"
    - $CKPT_DIR/ablated/lit_model.pth exists
    - $CKPT_DIR/ablated/ablation_manifest.json exists with non-empty zeroed_parameters list
    - `python -c "import json; m=json.load(open('$CKPT_DIR/ablated/ablation_manifest.json')); print(len(m['zeroed_parameters']), 'params zeroed')"`

    **Step 2 — Perplexity comparison (EVAL-01):**
    ```bash
    python -m safemoe evaluate --original $CKPT_DIR --ablated $CKPT_DIR/ablated
    ```
    Expected output table:
    ```
    Split          Original   Ablated    Delta
    D_std          XX.X       XX.X       small positive or near-zero
    D_harmful      XX.X       MUCH_HIGHER  significantly positive
    ```
    Key signal: D_harmful delta >> D_std delta. This proves isolation.
    Check $CKPT_DIR/results.json exists and has expected structure.

    **Step 3 — Routing attribution (EVAL-02):**
    ```bash
    python -m safemoe evaluate --original $CKPT_DIR --routing
    ```
    Expected:
    - routing_harmful_frac_D_harmful > routing_harmful_frac_D_std
      (harmful-domain tokens preferentially activate theta_harmful experts)
    - $CKPT_DIR/routing_attribution.json exists
    - TensorBoard logs in $CKPT_DIR/runs/routing_analysis/ (if TensorBoard installed)

    **Step 4 — Mid-training ablation curve (EVAL-03):**
    Open TensorBoard on the training run directory:
    ```bash
    tensorboard --logdir out/safemoe-tinystories/
    ```
    Check for `ablated_val_ppl_D_std` and `ablated_val_ppl_D_harmful` curves alongside
    the regular val_loss curve. If training ran long enough, ablated_val_ppl_D_harmful
    should increase over steps (delta grows as isolation improves).
  </how-to-verify>
  <resume-signal>
    Type "approved" if all four steps produce the expected isolation signal.
    Or describe any failures (e.g., "D_harmful delta is near zero after 200 steps — model
    needs more training") and whether that is acceptable for Phase 4 completion.
  </resume-signal>
