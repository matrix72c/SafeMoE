Function `ablate(ckpt_dir: Path) -> None`:
1. Resolve ckpt_dir = Path(ckpt_dir).
2. Load model_config.yaml: `raw = yaml.safe_load((ckpt_dir / "model_config.yaml").read_text())`.
   Build SafeMoEConfig: strip nested dicts (litgpt saves some nested keys under "model" sub-key),
   use `SafeMoEConfig(**{k: v for k, v in raw.items() if not isinstance(v, dict)})`.
3. Build model: `model = GPT(config)` — do NOT use fabric.init_module or fabric.setup here,
   as this is a standalone checkpoint manipulation (no DDP wrapping, no prefix stripping needed —
   see Pitfall 2 in RESEARCH.md).
4. Load weights: `state = torch.load(ckpt_dir / "lit_model.pth", map_location="cpu", weights_only=False)`.
   `model.load_state_dict(state["model"])`.
5. Build registry: `registry = HarmfulParamRegistry(model, config)` — raw model, no unwrapping.
6. Get theta_harmful params WITH their names: iterate `registry.parameters_by_type('theta_harmful')`
   but we also need names. Use `{id(p): name for name, p in model.named_parameters()}` to map
   param id -> name before zeroing.
7. Collect manifest entries and zero in-place:
   ```python
   id_to_name = {id(p): n for n, p in model.named_parameters()}
   manifest_entries = []
   total_pre_norm = 0.0
   for p in registry.parameters_by_type('theta_harmful'):
       name = id_to_name[id(p)]
       pre_norm = p.data.norm().item()
       total_pre_norm += pre_norm
       manifest_entries.append({"name": name, "pre_ablation_norm": pre_norm})
       p.data.zero_()
   ```
8. Print summary table:
   ```python
   n_zeroed = len(manifest_entries)
   expert_indices = list(config.harmful_expert_indices)
   post_norm = sum(p.data.norm().item() for p in registry.parameters_by_type('theta_harmful'))
   print(f"\nAblation summary:")
   print(f"  Expert indices zeroed : {expert_indices}")
   print(f"  Parameters zeroed     : {n_zeroed}")
   print(f"  Total norm (before)   : {total_pre_norm:.6f}")
   print(f"  Total norm (after)    : {post_norm:.6f}")
   ```
9. Save ablated checkpoint:
   ```python
   ablated_dir = ckpt_dir / "ablated"
   ablated_dir.mkdir(parents=True, exist_ok=True)
   torch.save({"model": model.state_dict()}, ablated_dir / "lit_model.pth")
   shutil.copy(ckpt_dir / "model_config.yaml", ablated_dir / "model_config.yaml")
   ```
10. Save manifest:
    ```python
    manifest = {"zeroed_parameters": manifest_entries}
    (ablated_dir / "ablation_manifest.json").write_text(json.dumps(manifest, indent=2))
    print(f"  Manifest saved        : {ablated_dir / 'ablation_manifest.json'}")
    print(f"  Ablated checkpoint    : {ablated_dir / 'lit_model.pth'}\n")
    ```
