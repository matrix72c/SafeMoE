- Build `model = GPT(config)` — raw model, no fabric.setup.

Helper `_get_val_loaders(ckpt_dir: Path, config: SafeMoEConfig, data_mock=None) -> dict`:
- If data_mock is not None: return data_mock.val_dataloaders().
- Otherwise: read x/y from config (attributes `config.x` and `config.y` if they exist,
  else fall back to yaml fields); construct MultiDataLoader from safemoe.data.datamodule,
  call connect(tokenizer=None, batch_size=4, max_seq_length=config.block_size), setup(),
  return val_dataloaders().
- Returns {"D_std": DataLoader, "D_harmful": DataLoader}.

Function `evaluate_perplexity(original_ckpt_dir: Path, ablated_ckpt_dir: Optional[Path] = None,
                               data_mock=None, out_dir: Optional[Path] = None) -> dict`:
1. original_ckpt_dir = Path(original_ckpt_dir).
2. `model_orig, config = _load_model(original_ckpt_dir)`.
3. `fabric = L.Fabric(devices=1, accelerator="auto"); fabric.launch()`.
4. `model_orig = fabric.setup(model_orig)`.
5. `val_loaders = _get_val_loaders(original_ckpt_dir, config, data_mock)`.
6. For each split in ["D_std", "D_harmful"]:
   - `loader = fabric.setup_dataloaders(val_loaders[split])`
   - `loss = validate(fabric, model_orig, loader, max_iters=sys.maxsize, verbose=False)`
   - `ppl = math.exp(loss.item())`
   - Store in `orig_ppls[f"val_ppl_{split}"] = ppl`
7. If ablated_ckpt_dir provided:
   - `model_abl, _ = _load_model(Path(ablated_ckpt_dir))`
   - `model_abl = fabric.setup(model_abl)`
   - Repeat step 6 for model_abl -> `abl_ppls`
   - Compute delta: `delta = {k: abl_ppls[k] - orig_ppls[k] for k in orig_ppls}`
8. Print comparison table:
   ```
   Split          Original   Ablated    Delta
   D_std          XX.XX      XX.XX      +0.XX
   D_harmful      XX.XX      XXX.X      +XX.X
   ```
9. Build result dict: `{"original": orig_ppls, "ablated": abl_ppls, "delta": delta}`.
   If no ablated_ckpt_dir: `{"original": orig_ppls}`.
10. Write results.json to `(out_dir or original_ckpt_dir) / "results.json"`.
11. Return result dict.

Function `routing_attribution(ckpt_dir: Path, data_mock=None) -> dict`:

2. `fabric = L.Fabric(devices=1, accelerator="auto"); fabric.launch()`.
3. `model = fabric.setup(model)`.
4. `val_loaders = _get_val_loaders(Path(ckpt_dir), config, data_mock)`.
5. `harmful_indices_set = set(config.harmful_expert_indices)`.
6. `safemoe_layers = [m for m in model.modules() if isinstance(m, SafeMoELayer)]`.
